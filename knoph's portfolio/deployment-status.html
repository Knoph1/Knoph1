How to Deploy This Portfolio

Follow these steps to deploy this portfolio to GitHub Pages:

  1. Initialize a Git repository:
    2. git init
  2. Add all files to the repository:
    3. git add .
  3. Commit the changes:
    4. git commit -m "Initial commit"
  4. Add the remote origin (replace with your repo URL):
    5. git remote add origin https://github.com/Knoph1/Knoph-s-Portfolio.git
  5. Push the code to GitHub:
    6. git push -u origin main
  6. Enable GitHub Pages:
    • Go to your repository on GitHub.
    • Click on "Settings".
    • Scroll down to "GitHub Pages".
    • Select the "main" branch and save.
